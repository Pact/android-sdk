android-sdk
===========

The SDK for catalyze.io on the Android platform

Download
--------

Download [the latest JAR][1] or grab via Maven:

```xml
<dependency>
    <groupId>io.catalyze.sdk</groupId>
    <artifactId>android</artifactId>
    <version>(insert latest version)</version>
</dependency>
```
